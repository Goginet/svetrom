Я хочу разработать простое web приложение для яхт школ, которое будет демонстрировать положение парусов на яхте в зависимости от положения ветра и выводить каким курсом движеться яхта (бейдевинг, галфинд, бакштаг, фордевинг).

Требования:
- 2d вид яхты сверху 
- схематическое отображение парусов (стаксель и грот)
- фиксированное положение ветра
- на мобильных устройствах при повороте телефона будет так же поворачиваться яхта и паруса будут менять свое положение относительно ветра
- если акселерометр на устройстве недоступен то будет возможноть ручного поворота яхты с помощью стрелок


npm install
npm run dev

Теперь давай нарисуем контуры яхты относительно мачты, основные параметры:
- Угол входа (Entry Angle): Насколько острый нос. Геометрически это положение первой контрольной точки относительно носа.
- Мидель (Midship): Точка максимальной ширины. У современных яхт она смещена назад (на 60–70% длины от носа). У классических — находится посередине.
- Транец (Transom): Ширина кормы.
- сдвиг мачты относительно Миделья
